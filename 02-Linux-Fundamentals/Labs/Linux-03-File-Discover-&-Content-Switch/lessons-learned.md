# Lessons Learned

- Linux navigation is simple when the directory structure is understood clearly
- `find` is useful for locating files without manually browsing every folder
- `grep` is useful when searching for logs and configuration files
- Command output can become noisy if searches are run from too broad a directory
- `cat` is best for readable text files, but not unknown binary style files
- File discovery and content search are practical daily skills for troubleshooting and investigation work
