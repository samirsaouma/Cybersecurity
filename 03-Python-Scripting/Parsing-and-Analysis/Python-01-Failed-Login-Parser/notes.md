# Notes

## What the script does

- Opens the text file `sample_log.txt`
- Reads it line by line
- Checks whether each line contains `Failed login`
- Prints matching lines
- Counts the total number of matches

## Why this is useful

This is a simple example of how Python can help reduce noise in logs and pull out relevant security events.

## Main lesson

A small script can still be useful if it solves one clear problem.
