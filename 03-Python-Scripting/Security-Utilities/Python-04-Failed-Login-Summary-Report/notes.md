# Notes

## What the script does

- Opens the log file
- Reads it line by line
- Identifies failed login events
- Counts the total number of failed logins
- Extracts usernames
- Counts how many times each username appears
- Extracts and lists source IP addresses
- Prints a simple summary report

## New idea in this project

This project combines several earlier projects into one script.

## Why this is useful

This is a basic example of turning raw log entries into a short report that is easily digestible and coherent.
