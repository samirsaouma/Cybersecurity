# Notes

## What the script does

- Opens the log file
- Reads it line by line
- Checks for failed login entries
- Extracts the username from each matching line
- Stores usernames in a list
- Prints the results and a final count

## New idea in this project

This project focuses on extracting one useful field from the log entry rather than only printing the whole line.

## Why this is useful

This is an example of turning raw security events into a more digestible view of who is being targeted.
