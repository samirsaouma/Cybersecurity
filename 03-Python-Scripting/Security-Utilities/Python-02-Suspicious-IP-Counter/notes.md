# Notes

## What the script does

- Opens the log file
- Reads it line by line
- Checks for failed login events
- Extracts the source IP address
- Counts how many times each IP appears
- Prints a simple summary

## New idea in this project

This project adds counting logic with a dictionary instead of only filtering lines.

## Why this is useful

This is a basic example of turning raw events into something more meaningful for investigation.
