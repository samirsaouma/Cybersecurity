# Notes

## What the script does

- Opens the log file
- Reads it line by line
- Checks for failed login events
- Extracts the username from each matching line
- Counts each time the username appears
- Sorts usernames by the frequency of appearance
- Prints a summary in order of frequency

## New idea & Usefulness

This project adds sorting so the most frequently targeted usernames appear first, this is useful for analysts to easily see priority targets.

## Why this is useful

This is a simple example of turning repeated failed login data into a more useful priority view for analysis.
