**Technical Lessons**

- All devices must share the same subnet to communicate locally
- Switches operate at Layer 2 using MAC addresses
- IP addressing errors are the most common connectivity issue

**Troubleshooting Lessons**

- Always verify IP configuration first
- Test connectivity with ping
- Check physical connections if communication fails

**Security Lessons**

- Flat networks allow easy lateral movement
- Network segmentation reduces attack spread
- Understanding packet flow helps detection engineering later
