## Lessons Learned

**Technical:**

- Learned how to capture live network traffic using Wireshark.
- Learned how to apply and remove basic Wireshark filters.
- Learned how to navigate through frame information, including seeing port number and protocol behavior. 

**Security Importance:**

- Packet visibility and network oversight is essential for detection engineering.
- Understanding and filtering normal traffic is required to identify anomalies.
- Traffic analysis is a core cybersecurity skill.

**Improvements:**
- Learn how to analyze and examine suspicious traffic patterns.
- Learn how attackers use spoofing capabilities to hide packet information, and to blend in.
