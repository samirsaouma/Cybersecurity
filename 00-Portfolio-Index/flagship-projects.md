# Flagship Projects

This file highlights the strongest integrated projects in the repository.

## Current Flagship Direction

### SME Secure Branch Network

A full enterprise-style branch network design combining:

- VLAN segmentation
- VLSM
- OSPF
- STP
- DHCP relay
- ACL policy enforcement
- PAT internet access
- wired and wireless user groups
- centralized services

This project serves as the first major architecture anchor in the portfolio.

## What makes a project a flagship

A flagship project should combine:

- multiple technologies
- realistic design decisions
- structured verification
